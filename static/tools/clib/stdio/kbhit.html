kbhit
 

  原型：extern int kbhit(void);
  
  用法：#include <stdio.h>
  
  功能：检测按键
  
  说明：检测键盘是否有键按下。
        如果有键按下，则返回对应键值；否则返回零。
        kbhit不等待键盘按键。无论有无按键都会立即返回。
  
  举例：

      // kbhit.c
      
      #include <stdio.h>

      main()
      {
        int i=0;
        
        clrscr();
        
        while(!kbhit())
        {
          clrscr();
          printf("%05d",i++);
        }
        
        clrscr();
        printf("End.");
        
        getchar();
        return 0;
      }
